# Demo Video Script — Tesseract (2–3 min)

**Total target: ~2:30**

---

### 0:00 – 0:15 | Opening

> Show app launching into a blank note.

**Visual:** App boots → notes sidebar visible → click an existing note.

**Narrator:**
"Hi, this is Tesseract — a local-first, AI-enhanced markdown notebook. Everything you write lives as plain `.md` files on your machine. No cloud, no accounts."

---

### 0:15 – 0:40 | Notes & Editing

> Type a few lines of markdown in the editor. Create a new note.

**Visual:** Type headings, bold, a list. Then click the ✏️ button in the sidebar → type a title → a new blank note opens.

**Narrator:**
"Full markdown editing with auto-save — three seconds after you stop typing, it's written to disk. Create a new note with the plus button, switch between them instantly."

---

### 0:40 – 1:10 | Command Palette & Themes

> Press Ctrl+K, type "font", increase font size. Then search "theme" and open Pick Theme.

**Visual:** `Ctrl+K` → command palette appears → type "font" → select Increase Font Size → see text grow. `Ctrl+K` again → type "theme" → select "Pick Theme..." → theme picker dialog → click "Amber Glow" → the whole UI warms up.

**Narrator:**
"Press `Ctrl+K` for the command palette — adjust font size, toggle dark mode, or pick from several color themes. Each theme overrides the entire palette in real time."

---

### 1:10 – 1:40 | AI Companion & /write

> Switch to AI mode via the ActivityBar. Type a question. Then type /write.

**Visual:** Click AI icon in ActivityBar → AI sidebar slides in → type "Summarize this note" → AI responds. Then type `/write Make this more formal` → note content updates with a confirmation badge.

**Narrator:**
"Click the AI icon to open the AI sidebar. Ask questions about your current note. Or type `/write` followed by instructions — the AI sees the full note and rewrites it entirely, inserting the result directly into the editor."

---

### 1:40 – 2:10 | Council of Thought

> Switch to Council mode. Watch a brief debate.

**Visual:** Click Council icon → Council view appears → type or paste an idea → three personas respond sequentially → scroll through their responses.

**Narrator:**
"The Council of Thought. Pick any idea and three AI personas debate it — the Visionary argues potential, the Skeptic attacks flaws, and the Pragmatist focuses on execution. You watch the debate and make the final call."

---

### 2:10 – 2:30 | Autocomplete & Wrap

> Show autocomplete in action, then close.

**Visual:** Type the start of a sentence → grey completion text appears → press Tab to accept. Fade to black.

**Narrator:**
"Inline autocomplete as you type — `Alt+A` to toggle, `Tab` or `Ctrl+Space` to accept. That's Tesseract. Write, think, debate."

---

## Production Notes

| Segment         | What to prepare                                                    |
| --------------- | ------------------------------------------------------------------ |
| Opening         | Have a note pre-written with some markdown content                 |
| Command Palette | Reset font size before recording so change is visible              |
| Theme Pick      | Start on Default theme, switch to Amber Glow for contrast          |
| AI /write       | Have the AI key set; use a short, simple note for quick turnaround |
| Council         | Pre-type a short idea so you can hit send immediately              |
| Autocomplete    | Type a common phrase like "The main advantage of this approach is" |
| Audio           | Use a lapel mic or quiet room — voiceover can be recorded after    |
| Edits           | Cut pauses; speed up AI loading with a quick jump cut              |
